
## **Project 1**

### **Project1: 医学语言理解**

在 [MedBench](https://medbench.opencompass.org.cn/docs) 中任选一个医学语言理解的数据集，

（选项一）在 Huggingface 上选择一个 BERT 系列的模型，在该数据集上进行微调，并给出性能评估，并与 Benchmark 中已有的其它 BERT 模型进行比较。

（选项二）在本地部署一个 LLM，进行该任务的测试，并与 Benchmark 中已有的 BERT 系列模型进行性能比较。

---
### 数据集介绍
#### CMeEE
该数据集由北京大学计算语言学教育部重点实验室等机构创建，是一个面向中文医学文本的命名实体识别数据集。
```
元数据

数据集中包含以下信息
question：文本
answer: 答案，实体列表
示例

{

  "question": "因而儿童肺部病变较轻，肺内炎性病变吸收较快。"

  "answer": {

   "药物": [],

   "医疗设备": [],

   "医院科室": [],

   "微生物类": [],

   "身体部位": [],

   "医疗程序": [],

   "医学检验项目": [],

   "症状和体征": [],

   "疾病": ["肺部病变", "肺内炎性病变"]

  }

}
```

#### CMeIE
该数据集由郑州大学自然语言处理实验室等机构创建，是一个面向中文医学文本的实体关系抽取数据集。
```
元数据

数据集中包含以下信息
question：医学文本和对应的问题
answer：实体关系列表
示例

{

  "question": "慢性胰腺炎@ ###低剂量放射 自1964年起，有几项病例系列报道称外照射 (5-50Gy) 可以有效改善慢性胰腺炎患者的疼痛症状。慢性胰腺炎@从概念上讲，外照射可以起到抗炎和止痛作用，并且已经开始被用于非肿瘤性疼痛的治疗。 ",

  "answer":{

   "关系":[

     {

       "关系": "放射治疗",

       "头实体": "慢性胰腺炎",

       "尾实体": "外照射"

     },

     {

       "关系": "放射治疗",

       "头实体": "非肿瘤性疼痛",

       "尾实体": "外照射"

     }

   ]

  }

}
```

#### CHIP-CDEE
该数据集由医渡云（北京）技术有限公司创建，任务为从中文电子病历中挖掘出临床事件。
```
元数据

数据集中包含以下信息
question: 病历或者医学影像报告
answer: 答案，包含主体词、发生状态、描述词、解剖部位
示例

{

  "question": "建议住院治疗，患者遂于今日入我院，门诊以“左肺门肿块待查”收入我科住院治疗。",

  "answer": [

   {

    "主体词": "肿块",

    "发生状态": "不确定",

    "描述词": [],

    "解剖部位": ["左肺门"]

   }

  ]

}
```


#### CHIP-CDN
该数据集由医渡云（北京）技术有限公司创建，针对中文电子病历中挖掘出的真实诊断实体进行语义标准化。
```
元数据

数据集中包含以下信息
question: 医学文本
answer：答案
示例

{

  "question": "1型糖尿病性植物神经病变

   候选标准词：

     1型糖尿病性自主神经病

     钙代谢紊乱##脑电图异常

     恶性黑色素瘤

     原发性甲状腺功能减退##甲状腺囊肿##甲状腺功能减退

     身体未特指部位的神经损伤##身体未特指部位的血管损伤

     支气管扩张症合并感染##肺不张",

  "answer": "1型糖尿病性自主神经病"

}
```

#### CHIP-CTC
该数据集由同济大学生命科学与技术学院创建，针对中文临床试验筛选标准的描述句子进行类别分类，定义了44种筛选标准语义类别。
```
元数据

数据集中包含以下信息
question: 临床试验筛选标准
answer: 类别
示例

{

  "question": "既往患有骨髓增生异常综合征（Myelodysplastic syndrome，MDS）的患者，或者未定型的急性白血病患者。

  "answer": "Disease",

}
```

#### SMDoc
SMDoc是一个医学文本结构化数据集，用于评估模型从真实医学文书中提取特定实体的能力。
```
元数据

数据集中包含以下信息
question：医学文本

answer：结构化结果，包括一般健康状况、高血压病史、糖尿病病史、传染病史、预防接种史、手术外伤史、输血史、食物过敏史、药物过敏史

示例

{

  "question": "医疗报告:

   一般健康状况: 一般

   疾病史: 糖尿病病史,平素未监测,具体不详,否认高血压病史

   传染病史: 否认结核乙肝病史

   预防接种史: 随社会

   手术外伤史: 否认

   输血史: 否认

   食物过敏史: 否认

   药物过敏史: 否认",

  "answer": {"一般健康状况": "一般",

   "高血压病史": "否认",

   "糖尿病病史": "有",

   "传染病史": "否认",

   "预防接种史": "随社会",

   "手术外伤史": "否认",

   "输血史": "否认",

   "食物过敏史": "否认",

   "药物过敏史": "否认"}

}
```
